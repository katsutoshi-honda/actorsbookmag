URL INPUT
映画の公式サイト、IMDb、プレスリリースのURLを入力するだけでパイプラインが起動。スクレイピングツールが自動でテキスト・メタデータ・画像を取得します。複数URLの一括処理にも対応。
  • 入力形式 URL / テキスト / RSS
  • 対応サイト IMDb, Letterboxd, 公式サイト
  • 処理方式 非同期キュー処理

// 03 — SNS STRATEGY
INSTAGRAM
FEED + REELS + STORIES
映画スチールを使ったビジュアル重視の投稿。縦長サムネイル(9:16)を自動生成。キャプションは映画の見どころを短くまとめる。
  • 画像サイズ 1080×1350px
  • Caption 最大2200字
  • API Graph API v20
X / TWITTER
TWEET + THREAD
記事のハイライトをスレッド形式で投稿。OGP画像付きURL。映画関連ハッシュタグを自動付与。エンゲージメント分析も実装。
  • 文字数 280字×スレッド
  • 画像 OGP + カスタム
  • API X API v2
TIKTOK
SHORT VIDEO
映画トレーラーのクリップ+テキストオーバーレイで自動ショート動画生成。トレンド音楽との組み合わせ提案も。
  • 動画長 15〜60秒
  • 形式 9:16 縦動画
  • API TikTok API v2
